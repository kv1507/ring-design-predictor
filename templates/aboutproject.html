Ring Design Choice Prediction

A machine learning approach to predict whether a jeweler's party will choose a specific ring design.

Project Overview

This project leverages machine learning to predict whether a jewelry party will select a given ring design. The model considers various factors such as:

  • Number of Diamonds
  • Color Combinations
  • Shape of the Ring
  • Quantity Ordered
  • Type of Design

Data Preprocessing

  • Handling missing values using mean/mode imputation.
  • One-hot encoding categorical features like Party Name, Shape, and Color.
  • Scaling numerical values using MinMaxScaler to normalize features.
  • Using SMOTE-Tomek to balance the dataset and reduce bias.

Model Selection

Multiple machine learning models were tested, and hyperparameter tuning was applied:

  • Logistic Regression: Baseline model with moderate performance.
  • Random Forest: Improved accuracy but overfitting risk.
  • XGBoost: Best-performing model after hyperparameter tuning.

Hyperparameter tuning:

  • Learning rate optimized using GridSearchCV.
  • Number of estimators set to 150 for balance between accuracy and execution time.
  • Max depth adjusted to prevent overfitting.
